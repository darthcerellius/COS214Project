Detailed Description

The abstract Instruction class acts as an interface for the concrete commands. These commands are invoked by the PitCrew and execute instructions in the Car class. PitCrew will send a command to the Car through one of the child classes: TacticalDecision or EventResponse. Acts as the Command participants in the Instruction Command pattern.
