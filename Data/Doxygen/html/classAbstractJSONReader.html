Detailed Description

Parses a JSON string and constructs a JSON object.

A string is passed into a JSON reader (either a JSONRoot or JSONObjectHolder) when it is constructed. This automatically creates a JSON object. You can access methods as you would in JavaScript, through the [] operator.
