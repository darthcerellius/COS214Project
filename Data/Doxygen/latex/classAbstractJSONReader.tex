\doxysubsection{Detailed Description}
Parses a J\+S\+ON string and constructs a J\+S\+ON object. 

A string is passed into a J\+S\+ON reader (either a \mbox{\hyperlink{classJSONRoot}{J\+S\+O\+N\+Root}} or \mbox{\hyperlink{classJSONObjectHolder}{J\+S\+O\+N\+Object\+Holder}}) when it is constructed. This automatically creates a J\+S\+ON object. You can access methods as you would in Java\+Script, through the \mbox{[}\mbox{]} operator.
