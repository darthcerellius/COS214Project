\doxysubsection{Detailed Description}
\mbox{\hyperlink{classCommand}{Command}} object that executes part of the system. 

The \mbox{\hyperlink{classCommand}{Command}} class is the abstract class that allows concrete commands to be linked in a Chain of Responsibility. This class enables linking, while the children commands implement the execute function.
