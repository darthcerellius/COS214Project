\doxysubsection{Detailed Description}
Simulates a race event for the system. 

This class has three functions\+: qualify, practice and race. Practice is meant for the Friday practice day, qualify is meant to determine the qualifying position of the car, and race performs the actual race. Race also generates random events for the \mbox{\hyperlink{classPitCrew}{Pit\+Crew}} to handle.
